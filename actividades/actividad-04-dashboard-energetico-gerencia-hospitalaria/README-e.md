# Actividad 4: Dashboard energético para la gerencia hospitalaria

## Introducción

En la gerencia de proyectos sostenibles, la visualización de datos es una herramienta clave para el análisis y la toma de decisiones. Esta actividad busca que el estudiante diseñe un dashboard energético que integre los modelos de generación y consumo desarrollados en actividades anteriores, presentando la información de manera clara, útil y orientada a la gerencia hospitalaria.

## Objetivo

Diseñar e implementar un dashboard energético en Python que integre datos de generación solar y consumo hospitalario, facilitando la toma de decisiones estratégicas para la gerencia del hospital.

## Desarrollo teórico

Un dashboard de gerencia es una herramienta de inteligencia operacional que permite visualizar indicadores clave de desempeño (KPIs) de manera consolidada. En el contexto del ecosistema hospitalario solar, los KPIs relevantes incluyen la energía generada, la energía consumida, el porcentaje de autosuficiencia solar, el ahorro económico estimado y las alertas de sobredemanda. Desde la perspectiva de proyectos, el dashboard convierte datos técnicos en información gerencial comprensible.

## Entregables esperados

- Dashboard funcional en Python (usando librerías como Matplotlib, Plotly o Dash).
- Visualización integrada de generación solar y consumo hospitalario.
- Indicadores clave de desempeño energético (KPIs).
- Documentación del diseño y funcionalidad del dashboard.

## Conclusión

A través de esta actividad se desarrollan competencias clave en análisis, síntesis y comunicación de datos para la gestión estratégica de proyectos hospitalarios sostenibles.
